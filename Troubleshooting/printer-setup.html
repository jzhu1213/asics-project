Printer Setup for Windows/PC

Method 1

Step 1: In the Windows search bar, open "Run" and type in "\\aac-boston-dc1".

If you open the "Run" application, a small window in the bottom left corner of your screen will appear. Type in "\\aac-boston-dc1" and press Enter.

Step 2: Right click the printer(s) you want to add and Connect.

Right click on the printer you want to use and hit connect. It will take a few seconds, then you can start printing.

Method 2

Step 1: Open the Start menu and select Settings.

You can find settings by either searching for it in the Windows search baror clicking the Windows button then Settings icon.

Step 2: Navigate to Printers and Scanners and select Add device. Choose Add manually.

Click "Bluetooth & devices" or use the search bar in the top left corner, then select "Add device" in the top right corner.

After waiting for a few seconds, there should be an option that says "The printer that I want isn't listed", allowing you to "Add manually".

Step 3: Add your printer using an IP address or hostname. Input the TCP/IP Device IP address and name your printer. Check the Query box.

Select the option bubble to "Add a printer using an IP address or hostname" then click "Next".

Change the Device type to "TCP/IP Device", enter the IP address, and change the name of the printer to whatever you'd like.

Make sure to check the "Query the printer and auotmatically select the driver to use" box.

Step 4a: Select the Canon Generic Plus PCL6 driver user that driver.

If the "Canon Generic Plus PCL6" driver does not show up, move to step 4b.

Otherwise, click "Next" and "Use the driver that is currently installed".

Move on to step 5.

Step 4b: Add the Canon 5740 driver.

Download the driver "here" and keep it somewhere accessible in your files.

Click "Have disk" and "Browse", then select the driver fiel you downloaded and click "Open" then "OK".

Move on to step 5.

Step 5: Name the printer and Finish.

After clicking "Next", give the printer whatever name you'd like, share the printer on the network or not, and click "Finish".

Printer Setup for Mac

Step 1: Click the Apple menu and select System Preferences.

Step 2: Navigate on Printers & Scanners and Add New Printer.

Step 3: Enter the IP Address of the printer you want to add and name it. Make sure the___ is LPD and ___ is Canon.

Step 4: Click to Printers & Scanners to add a new printer.
