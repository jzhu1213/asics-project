PC Imaging & OS Updates

Complete guide for reimaging PCs and setting up databases for ASICS Boston operations

New PC Imaging Procedure

Pre-Imaging Checklist

  • Verify PC hardware specifications meet requirements
  • Ensure all necessary cables and peripherals are available
  • Check network connectivity in imaging location
  • Prepare ASICS image USB drive or network boot

Step-by-Step Imaging Process

  1. Boot from ASICS Image:
    • Insert ASICS image USB drive or connect to network boot
    • Press F12 during startup to access Boot Menu
    • Select USB drive or network boot option
    • Wait for imaging environment to load
  2. Select Imaging Options:
    • Choose "Standard ASICS Image" from menu
    • Select target drive (usually C:)
    • Confirm "Clean Install" option
    • Begin imaging process
  3. Monitor Progress:
    • Imaging typically takes 20-30 minutes
    • Do not interrupt power or remove USB drive
    • Wait for completion message
  4. Post-Imaging Setup:
    • Remove USB drive when prompted
    • Allow system to restart automatically
    • Complete Windows initial setup
    • Log in with temporary credentials

Required Software Updates

  1. Windows Updates:
    • Run Windows Update multiple times until no updates remain
    • Restart computer after each major update
    • Verify Windows version is current
  2. Application Updates:
    • Update Microsoft Teams to latest version
    • Update Google Chrome browser
    • Update TeamViewer if installed
    • Verify all ASICS applications are current

PC Refresh Procedure

Data Backup (If Required)

  1. Identify critical user data to preserve
  2. Backup to network drive or external storage
  3. Document user preferences and settings
  4. Verify backup integrity before proceeding

Refresh Process

  1. Follow same boot and imaging steps as new PC
  2. Choose "Refresh" option instead of "Clean Install" if available
  3. Restore backed up data after imaging
  4. Reconfigure user-specific settings

Database Setup on Existing Machines

Prerequisites

  • Verify machine meets database requirements
  • Ensure adequate storage space (minimum 50GB free)
  • Check network connectivity to database server
  • Obtain database credentials and connection strings

Database Installation Steps

  1. Install Database Client:
    • Download appropriate database client software
    • Run installer with administrative privileges
    • Configure connection settings
    • Test connection to database server
  2. Configure Applications:
    • Update application configuration files
    • Set database connection parameters
    • Test application database connectivity
    • Verify data access permissions
  3. Security Setup:
    • Configure database encryption if required
    • Set up user authentication
    • Test backup and recovery procedures
    • Document connection details securely

Common Issues & Solutions

Imaging Problems

  • Boot Menu Not Appearing: Try different function keys (F2, F8, F10, F12)
  • USB Not Detected: Try different USB port or recreate bootable drive
  • Imaging Fails: Check hard drive health and try different imaging method
  • Slow Imaging: Verify USB drive speed and try USB 3.0 port

Post-Imaging Issues

  • Windows Won't Boot: Check boot order in BIOS settings
  • Missing Drivers: Install manufacturer drivers from support website
  • Network Issues: Update network adapter drivers
  • Application Errors: Reinstall problematic applications

Database Connection Issues

  • Connection Timeout: Check network connectivity and firewall settings
  • Authentication Failed: Verify credentials and user permissions
  • Slow Performance: Check network bandwidth and database server load
  • Data Corruption: Run database integrity checks and restore from backup
