Asset Tracking Overview

Manage assets in the ANA Asset Management Overview dashboard.

ANA Asset Management Overview Dashboard

Asset Status Overview

The Asset Status Overview tab shows all the assets currently under ANA. Navigate through each model category to see the assets assigned to each model, including Lenovo ThinkPad, Macbooks, and handheld/mobile devices.

Individual Asset Details

Search for an asset and select it to view its General, Financial, and Assignment information. You can see the state, serial number, depreciation, person assigned to, and more of each device.

Assigning Assets

When adding a "New" asset to the database, the same page above will be shown with no fields filled in. Input the appropriate information in each field then "Submit" the asset to the database. Make sure the General and Financial information is correct and all required fields are filled in.
